## Descripción del proyecto

Este proyecto consiste en la recreación de una sección principal de un sitio web real utilizando únicamente HTML semántico y CSS puro, sin el uso de frameworks ni JavaScript.

Se implementaron buenas prácticas de desarrollo web, incluyendo estructura semántica, organización del código y estilos modernos similares al diseño de Apple.

---

## Tecnologías utilizadas

- HTML5 (etiquetas semánticas)
- CSS (archivo externo)
- Google Fonts (tipografía Roboto)

---

## Características implementadas

- Uso de etiquetas semánticas: `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`
- Diseño inspirado en Apple
- Barra de navegación centrada y estilo moderno
- Buscador interactivo con efecto hover
- Secciones organizadas (hero, features, imagen, CTA)
- Formulario de contacto con 3 campos
- Uso de imagen con atributo `alt` descriptivo
- Uso de variables CSS
- Uso de selectores (elemento, clase y pseudo-selectores)
- Diseño responsive (adaptado a dispositivos móviles)
